Camera Ready Revision Checklist

1. Submission Package (Mandatory)

  • Format: The submission must be a single ZIP file.
  • Contents: The ZIP file must contain both the compiled PDF and the complete LaTeX source files (including .tex, .bib, style files, and all figure files).

2. Header & Metadata (Mandatory)

  • Publication Date: Ensure the Year and Month in the paper header are updated to match the acceptance date (or the date specified by the Action Editor).
  • Title & Abstract: The title and abstract in the PDF must exactly match the metadata entered in the OpenReview system (as this will be used for indexing).

3. De-anonymization

  • Author Names: Ensure the \anonymous option is removed so that author names and affiliations are visible.
  • Acknowledgments: Un-hide the Acknowledgments section (funding, reviewer thanks) if it was previously hidden for blind review.

4. Formatting & Assets

  • High-Quality Figures: Ensure all images are vector graphics (PDF/EPS) or high-resolution bitmaps (300dpi+).
  • Bibliography: Check for and fix any undefined references (e.g., [?] or (??)).

Please refer to our writing sample: https://github.com/JmlrOrg/jmlr-style-file/blob/master/sample.pdf
